# Website Pembelajaran Fotosintesis

Multi-halaman berisi materi, video, kuis 10 soal, rangkuman, dan profil.

## Struktur
- index.html
- materi.html
- kuis.html
- rangkuman.html
- profil.html
- assets/css/styles.css
- assets/js/main.js

## Cara pakai
1. Letakkan folder `website-fotosintesis` di lokal.
2. Upload ke GitHub (push repo) atau Netlify (drag & drop folder).
3. Untuk GitHub Pages: aktifkan Pages -> source main / root.
